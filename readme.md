# 365admin Version 1

## Purpose

The purpose of this kitchen is to setup wellknown structures for the 365admin project.

## Folder Structure

The folder structure is as follows:

```
    "root"
    ├── .koksmat
    │   ├── servers
    │   ├── sessions
    │   ├── workdir
    ├── workspaces
    │   ├── .koksmat
    │   │   ├── tenants
    │   │   |  ├── <tenant-name>
    │   │   |  |  ├── <connection-A>
    │   │   |  |  ├── <connection-B>
    
```
